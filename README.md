# AI政企瞭望系统

## 项目简介

AI政企瞭望系统是一个基于Flask框架开发的全景数据监测与智能分析平台，旨在为政府和企业提供全面的数据采集、分析和可视化服务。系统集成了爬虫技术、AI分析能力和数据可视化功能，实现了从数据采集到智能分析的全流程管理。

### 核心功能

- **数据采集**：支持多种数据源的自动采集和定时任务管理
- **AI分析**：基于人工智能技术对采集的数据进行深度分析和预测
- **数据可视化**：通过3D地球模型、图表等多种方式直观展示数据分布和趋势
- **实时监控**：实时监测系统运行状态和数据指标
- **深度数据采集**：针对特定领域进行深度数据挖掘和分析

## 技术栈

- **后端**：Python 3.8+, Flask 2.0+
- **前端**：HTML5, CSS3, JavaScript, Tailwind CSS
- **数据可视化**：ECharts 5.4+, ECharts-GL 2.0+
- **数据存储**：SQLite（可扩展为MySQL/PostgreSQL）
- **AI技术**：集成多种AI模型进行数据分析和预测
- **爬虫技术**：自定义爬虫框架，支持多源数据采集

## 安装与运行

### 环境要求

- Python 3.8 或更高版本
- pip 包管理工具

### 安装步骤

1. **克隆项目**

```bash
git clone <项目仓库地址>
cd ai-liaowangweb-app
```

2. **安装依赖**

```bash
pip install -r requirements.txt
```

3. **初始化数据库**

```bash
python init_db.py
```

4. **运行系统**

```bash
python run.py
```

5. **访问系统**

打开浏览器，访问 `http://127.0.0.1:5001` 即可进入系统。

## 系统架构

### 前端架构

- **响应式设计**：适配不同屏幕尺寸
- **组件化开发**：模块化设计，便于维护和扩展
- **数据可视化**：使用ECharts实现多种图表和3D地球模型
- **实时数据**：支持数据实时更新和监控

### 后端架构

- **MVC模式**：模型-视图-控制器架构
- **服务层**：业务逻辑封装，提供统一接口
- **数据层**：ORM映射，支持多种数据库
- **API设计**：RESTful API设计，便于前后端分离

## 主要功能模块

### 1. 数据采集模块

- **爬虫管理**：创建、编辑、删除爬虫任务
- **定时采集**：设置采集频率和时间
- **多源数据**：支持从多种数据源采集数据
- **数据清洗**：自动清洗和标准化采集的数据

### 2. AI分析模块

- **数据分析**：对采集的数据进行深度分析
- **趋势预测**：基于历史数据预测未来趋势
- **智能推荐**：根据分析结果提供智能建议
- **报告生成**：自动生成分析报告

### 3. 数据可视化模块

- **3D地球模型**：全球数据分布可视化
- **仪表盘**：关键指标实时监控
- **趋势图表**：数据变化趋势分析
- **地理分布**：数据地理分布展示

### 4. 深度数据采集模块

- **特定领域**：针对特定领域进行深度数据采集
- **多维度分析**：从多个维度分析数据
- **关联分析**：分析数据之间的关联关系
- **异常检测**：自动检测数据异常
